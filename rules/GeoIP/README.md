# GeoIP Rules

最后更新: 2026-07-22 12:10:46

## 目录说明

| 目录 | 说明 |
| --- | --- |
| `txt` | 纯文本 CIDR 列表 |
| `json` | sing-box rule-set 源格式 |
| `srs` | sing-box 编译后的二进制 rule-set |
| `yaml` | mihomo rule-set 源格式（ipcidr payload） |
| `mrs` | mihomo 编译后的二进制 rule-set |

## 已选择的 GeoIP 分类

```text
cloudflare cloudfront google twitter facebook private netflix telegram
```
